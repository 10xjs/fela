5. API Reference

API Reference

Last but not least this chapter ships the API reference in detail. Despite explaining every single method its arguments and return value, you will also find some tips on some pages.

Top-Level API

fela

  • createRenderer([config])
  • combineRules(...rules)
  • enhance(...enhancers)

fela-dom

  • render(renderer, mountNode)

Renderer API

  • .renderRule(rule, [props])
  • .renderKeyframe(keyframe, [props])
  • .renderFont(family, files, [properties])
  • .renderStatic(style, [selector])
  • .renderToString()
  • .subscribe(listener)
  • .clear()
